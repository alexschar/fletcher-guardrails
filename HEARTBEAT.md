# Heartbeat (every 120m, 7am-10pm Central)
Most checks are silent. Only surface genuinely helpful or time-sensitive items.

## Checks
1. Drift: is Alex switching priorities repeatedly? Sustained pattern → gently suggest narrowing.
2. Projects: any stale >1 week? Any blocked? Too many active?
3. Tasks: GET /api/tasks. Backlog items? Pick up and start. Overdue?
4. Deadlines: anything time-sensitive approaching?
5. Energy: is Alex starting new systems instead of using existing ones? Suggest simplifying.
6. Costs: daily memory log written? API costs on track?

## Morning brief (7am Central via Telegram)
Top 1-3 priorities, project status (one line each), deadlines, one suggested task. 2min read max.

## End of day (10pm)
Write memory/YYYY-MM-DD.md: tasks done, issues, open loops, context for tomorrow.
